# MCP로 구성하는 AI 기반 모바일 앱 보안 감사

이 deck은 사용자가 제공한 원문 구성안을 바탕으로 `present-anything` 프로젝트에 추가한 상세 발표 자료입니다.

## 편집 원칙

- 원문 흐름은 최대한 유지
- 직접적인 특정 OS 언급은 제거하거나 일반화
- 발표 자료에 맞게 문장과 표 구조를 정리
- 내부 경로나 구현 냄새 나는 표현은 줄이고 설명 중심으로 구성

## 실행

```bash
npm run dev:mobile-audit-mcp-origin
```

## 빌드

```bash
npm run build:mobile-audit-mcp-origin
```
