# Windows 환경에서 MCP로 구성하는 AI 기반 모바일 앱 보안 감사 - Extended

이 deck은 기존 `mobile-audit-mcp-windows` 요약본보다 원문 구성안에 더 가깝게 확장한 상세 버전입니다.

## 특징

- 약 30장 이상의 상세 흐름
- 원문의 개념 설명, 환경 구축, 도구별 적용, 시스템 예시, Skill 설계, 운영 통제를 더 많이 보존
- 요약 발표보다 워크숍, 내부 세미나, 상세 설명 세션에 적합

## 실행

```bash
npm run dev:mobile-audit-mcp-extended
```

## 빌드

```bash
npm run build:mobile-audit-mcp-extended
```
