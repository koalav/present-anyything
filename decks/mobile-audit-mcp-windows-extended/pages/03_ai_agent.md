---
layout: two-cols

# 일반 LLM

- 설명, 요약, 코드 생성에 강함
- 외부 도구 실행 능력은 없음
- 파일, 디바이스, 프록시를 직접 조작하지 못함

::right::

# AI Agent

- 목표를 작업 단위로 분해
- 필요한 Tool을 선택
- 실행 결과를 읽고 다음 행동 결정
- 최종 결과를 Finding, 표, 보고서로 정리

---

# 보안 감사에서의 Agent 역할

- 자연어 요청을 ADB, Frida, Ghidra 작업으로 변환
- 로그와 코드에서 이상 징후를 요약
- 증거를 Finding 후보에 연결
- MASVS 기준으로 보고서 초안 작성
