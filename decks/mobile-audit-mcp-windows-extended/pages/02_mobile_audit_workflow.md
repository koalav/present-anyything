# 2. 모바일 앱 감사의 작업 구조

---

# 모바일 앱 감사는 복합 작업이다

1. 감사 범위와 분석 대상을 확정한다.
2. APK, IPA, 소스 코드와 기본 메타데이터를 확보한다.
3. Manifest, 권한, 컴포넌트, 코드 패턴을 정적으로 살핀다.
4. 디바이스에서 실행해 화면, 로그, 함수 흐름을 수집한다.
5. `.so`, WebView, API 트래픽까지 연결해 본다.
6. 마지막에 증거를 정리하고 보고서 초안을 만든다.

---

# 왜 어렵나

- 도구가 많고 명령어가 흩어져 있다
- 정적, 동적, 네트워크 결과가 따로 논다
- 증거 저장 규칙이 없으면 재검증이 어렵다
- 보고서 품질이 감사자 경험에 크게 의존한다
