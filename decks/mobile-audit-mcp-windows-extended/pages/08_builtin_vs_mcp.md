# 일반 CLI Agent의 기본 도구

- 파일 읽기
- 파일 검색
- 파일 편집
- 셸 실행
- 패치 적용
- 웹 조회
- 작업 관리

::right::

# MCP로 추가되는 전문 도구

- Android 디바이스 제어
- JADX 정적 분석
- Frida 런타임 후킹
- Ghidra 바이너리 분석
- Burp, ZAP, Chrome DevTools 연동

# 구분 원칙

- 기본 도구: Agent 자체의 작업 수행 능력
- MCP 도구: 외부 전문 시스템 접근 능력
- Skill: 두 계층을 감사 절차로 묶는 방식
