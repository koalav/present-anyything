# MobSF, Semgrep, CodeQL의 위치

| 도구 | 역할 | 해석 원칙 |
|---|---|---|
| MobSF | APK 초벌 정적 분석 | 결과를 그대로 확정하지 않음 |
| Semgrep | 반복 가능한 룰 기반 탐지 | 오탐과 중복 제거 필요 |
| CodeQL | 데이터플로우 중심 질의 | 추가 검증과 코드 맥락 필요 |
