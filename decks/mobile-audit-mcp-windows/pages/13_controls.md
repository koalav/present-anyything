# 13. 운영 통제

---

# 반드시 통제해야 할 것

- 과도한 Tool 권한
- 셸 명령 오남용
- 민감정보의 모델 컨텍스트 유입
- 웹 페이지와 이슈 본문 기반 프롬프트 인젝션
- 악성 MCP Server 설치
- 승인 없는 능동 스캔

---

# 권장 통제 원칙

| 영역 | 통제 방식 |
|---|---|
| 대상 범위 | allow-list 기반 |
| 위험 작업 | 승인 게이트 |
| 비밀값 | 자동 redaction |
| 도구 권한 | 최소 권한 토큰 |
| 로그 | Tool Call audit log 보존 |
| 실행 환경 | 테스트 환경 격리 |
| 결과 확정 | Human-in-the-loop |

<div class="mt-8 p-4 rounded bg-slate-100">
속도를 높이는 것보다 중요한 것은
<b>잘못된 실행이 빠르게 확산되지 않게 막는 것</b>이다.
</div>
