# 최종 요약

- MCP는 다양한 보안 도구를 일관된 방식으로 연결한다
- Tool은 실제 실행 단위이며 권한 통제가 필요하다
- Skill은 감사 절차와 산출물 품질을 표준화한다
- Agent는 여러 증거를 엮어 Finding 후보를 만든다
- 최종 검증과 책임은 감사자에게 남아 있어야 한다
