# 7. Frida와 런타임 검증

---

# Frida MCP의 역할

- 함수 호출 인자와 반환값 관찰
- 런타임에 복호화되는 값 확인
- 인증 흐름과 암호화 흐름 추적
- 난독화된 코드의 실제 동작 확인
- 네이티브 함수 호출과 Java bridge 연결 보조

---

# 동적 분석에서 중요한 점

- 정적 분석에서 본 후보를 검증하기 위한 도구로 사용
- 테스트 계정과 승인된 환경만 사용
- 토큰, 키, 비밀번호는 마스킹한 메타데이터만 남김
- 후킹 스크립트 실행 전 대상 함수와 클래스명을 검토

```text
예시 요청:
LoginManager.validateUser 호출 인자와 반환값을 추적하고
성공/실패 흐름만 짧게 요약해줘.
민감값 원문은 출력하지 마.
```
